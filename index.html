Introducción

Este ERP busca gestionar inventario, ventas, facturación y reportes para una tienda de ropa.

Requisitos

  • Gestión de inventario
  • Control de ventas
  • Facturación automática
  • Reportes en tiempo real

Arquitectura del Sistema

El sistema estará basado en arquitectura cliente-servidor con base de datos MySQL.

Manual de Usuario

El sistema cuenta con módulos de autenticación, inventario y ventas. Usa el menú de navegación superior para acceder a cada sección.
